# ng-blog-example-app

Example App

If you want to test the app localy you should do the following:

1. clone or download the repo
2. open the terminal and run `cd ng-blog-client/`
3. `npm install`
4. `ng serve`
5. in another terminal window run `cd ng-blog-api/`
6. `npm install`
7. `mongorestore dump/`
8. `node bin/www`

### Client:

- Angular v6
- AngularCli
- AngularMaterial


### Api: 

- Mongo DB
- NodeJS
- ExpressJS
- Passport
